spzoom

Simple image zoomer for jQuery

Usage

  1. <script type="text/javascript" src="jquery.spzoom.js"></script>
  2. <link rel="stylesheet" type="text/css" href="jquery.spzoom.css">
  1. <a href="46_0923.jpg" class="spzoom" title="medal sygnowany S.D. (Sebastian Dadler) wybity w 1636 roku">
  2.     <img src="46_0923_thumb.jpg" alt="thumb"/>
  3. </a>
  1. $(function() {
  2.     $(".spzoom").spzoom();
  3. });

Options

option default description
width 250 Zoom window width
height 250 Zoom window height
position 'right' Position of the zoom window relative to the thumb. Can be one of top/right/bottom/left
margin 20 Additional space between thumb and zoom window
cursor 'crosshair' Cursor style. null to leave alone
title 'bottom' Image title position (title is read from anchor title attr.). Set to 'off' to not display title. Valid values are: top/bottom/off
error 'Loading error!' Message to display in case of image loading error. Can be null - means do not show error.
min_tracker 15 Min. dimension tracker window can have. Smaller tracker won't be drawn (only cursor)
